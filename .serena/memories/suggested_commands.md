# Suggested Commands

## Running Tests

```bash
# Install dependencies
composer install

# Run PHPUnit (requires ClickHouse running locally)
./vendor/bin/phpunit
```

## Git Commands (macOS/Darwin)
```bash
git status
git add .
git commit -m "message"
git push
git pull
git log --oneline -10
git diff
```

## File System (macOS/Darwin)
```bash
ls -la
find . -name "*.php"
grep -r "pattern" src/
```
